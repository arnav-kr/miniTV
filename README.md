## About

Downloader For Amazon minitv

## Features
* Download Web Series/Episodes from Amazon minitv

## Install

**Via NPM:**

```bash
npm install minitv
```

**Via Yarn:**

```bash
yarn add minitv
```

**Via PNPM**
  
```bash
pnpm install minitv
```
